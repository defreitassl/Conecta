# Programa Connecta

Workspace de planejamento, documentação, apresentações e automações do Programa Connecta.

## Estrutura

- `apresentacoes/` - apresentações executivas em HTML e seus ativos visuais em `assets/branding/`.
- `documentos/` - matriz principal, planos e notas gerais.
- `Conecta Discord/` - documentação estruturada da operação no Discord.
- `discord-server-setup/` - scripts e configuração técnica do servidor Discord.
- `output/` - capturas e artefatos gerados durante validações.

## Pontos de entrada

- `apresentacoes/apresentacao-matriz-connecta.html`
- `apresentacoes/apresentacao-discord-conecta.html`
- `documentos/Matriz Connecta V2.md`
